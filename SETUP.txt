🚀 RADIO STATION REQUEST SYSTEM - LOCAL SETUP GUIDE
=================================================

PREREQUISITES:
-------------
1. PostgreSQL installed and running
2. Node.js installed (v14 or higher)

SETUP STEPS:
-----------

1. CREATE DATABASE:
   Open PostgreSQL command line (psql) and run:

   CREATE DATABASE radio_station_requests;

2. UPDATE DATABASE CONNECTION:
   Edit backend/.env file and update DATABASE_URL with your PostgreSQL credentials:

   DATABASE_URL=postgresql://YOUR_USERNAME:YOUR_PASSWORD@localhost:5432/radio_station_requests

3. SETUP BACKEND:
   cd backend
   npm install
   npm run setup-db
   npm run dev

   Backend will run on: http://localhost:5000

4. SETUP FRONTEND (in a new terminal):
   cd frontend
   npm install
   npm start

   Frontend will run on: http://localhost:3000
